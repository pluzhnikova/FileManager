В менеджере можно:
	Работать с файлами:
		1)Переместить(Выделить объект + Файл->Переместить + Выбрать папку + Файл->Переместить)
		2)Копировать(Выделить объект + Файл->Копировать)
		3)Вставить(Выделить объект + Файл->Вставить)
		4)Переименовать(Выделить объект + Файл->Переименовать + Написать новое имя с расширением + ОК)
		5)Удалить(Выделить объект + Файл->Удалить)
		6)Архивировать(Выделить объект + Файл->Архивировать)
		7)Разархивировать(Выделить объект + Файл->Разархивировать + Ввести имя файла с расширением + ОК)
			*Если папка, то имя создаётся автоматически

	Работать с дизайном:
		1) выбрать Тему
		2) выбрать Цвет области с файлами
		3) выбрать Шрифт в области с файлами
		4) выбрать Вид области с файлами
	Если после выбора настроек нажать "Использовать настройки в следующий раз" - настройки сохранятся.
	
	Скачать файл
	Перейти к файлу или папке(Ввести в тектовое поле + Перейти)
	Перейти к Дискам
	Искать файл по регулярному выражению - панель справа
